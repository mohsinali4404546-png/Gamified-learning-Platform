# 🎮 Gamified Learning Platform (Points & Badges)

A modern and interactive **Gamified Learning Platform** developed as a university project. The platform applies gamification concepts such as **points, badges, leaderboards, quizzes, and progress tracking** to make learning more engaging and motivating for students.

---

## 📌 Project Overview

The Gamified Learning Platform encourages learners to complete lessons, participate in quizzes, and earn rewards. By incorporating game-like elements into education, the platform improves student engagement and creates a more enjoyable learning experience.

---

## ✨ Features

* 🔐 User Registration & Login
* 📚 Learning Modules
* 📝 Quiz & Assessment System
* ⭐ Points & Rewards
* 🏅 Badge System
* 📈 Progress Tracking
